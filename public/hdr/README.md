# HDRI Environment Files

This directory contains HDRI environment maps for soft studio lighting.

## Recommended File

- `studio.hdr` - Soft studio lighting environment (optional)

## HDRI Sources

You can find free HDRI environments at:

- Poly Haven (https://polyhaven.com/hdris)
- HDRI Haven (https://hdrihaven.com/)
- AmbientCG (https://ambientcg.com/)

## Usage

The HDRI is used by the model-viewer component to provide:

- Soft, realistic lighting
- Environment reflections
- Ground plane reflections (if supported)

## File Format

- **Format**: HDR or EXR
- **Resolution**: 2K or 4K recommended
- **Type**: Studio/indoor lighting preferred for product visualization

## Optional

If no HDRI is provided, model-viewer will use its default environment lighting.
