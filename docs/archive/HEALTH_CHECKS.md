# Автоматические проверки работоспособности

Система автоматических проверок для предотвращения поломок проекта.

## Доступные скрипты

### 1. Быстрая проверка (рекомендуется)

```bash
npm run health:quick
# или
node scripts/quick-check.js
```

- Проверяет наличие критических файлов
- Проверяет синтаксис
- Не запускает полную сборку
- ⚡ Быстро (1-2 секунды)

### 2. Полная проверка

```bash
npm run health
# или
node scripts/health-check.js
```

- Полная проверка всех компонентов
- Запускает сборку проекта
- Проверяет импорты и зависимости
- ⏱️ Медленнее (30-60 секунд)

### 3. Автоматическая проверка

```bash
node scripts/auto-verify.js
```

- Легкая проверка после изменений
- Используется AI-ассистентом автоматически

## Когда использовать

- **После каждого изменения кода** - быстрая проверка
- **Перед коммитом** - полная проверка
- **При ошибках** - полная проверка для диагностики

## Что проверяется

✅ Наличие критических файлов  
✅ Синтаксис кода  
✅ Импорты и зависимости  
✅ Ошибки компиляции  
✅ Структура проекта

## Автоматический запуск

AI-ассистент автоматически запускает проверки после изменений кода для предотвращения поломок.
