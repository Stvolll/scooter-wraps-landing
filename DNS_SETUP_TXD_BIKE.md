# Настройка DNS для txd.bike

## Текущий статус

- Домен добавлен в Vercel проект: `scooter-wraps-landing`
- Статус: Ожидает настройки DNS

## Вариант 1: A-запись (Рекомендуется)

Добавьте следующую запись у вашего DNS-провайдера:

### Запись A (IPv4)

```
Тип: A
Имя/Хост: @ или txd.bike
Значение/IP: 76.76.21.21
TTL: 3600 (или автоматически)
```

### Запись для www поддомена (опционально)

```
Тип: CNAME
Имя/Хост: www
Значение: cname.vercel-dns.com
TTL: 3600 (или автоматически)
```

## Вариант 2: Изменение Nameservers

Если вы хотите управлять DNS через Vercel, измените nameservers у регистратора домена:

### Nameservers Vercel:

```
ns1.vercel-dns.com
ns2.vercel-dns.com
```

### Текущие nameservers:

```
dns1.registrar-servers.com
dns2.registrar-servers.com
```

## Инструкции по настройке у популярных регистраторов:

### Namecheap

1. Войдите в аккаунт
2. Перейдите в Domain List
3. Нажмите Manage рядом с txd.bike
4. Перейдите в Advanced DNS
5. Добавьте A Record:
   - Host: @
   - Value: 76.76.21.21
   - TTL: Automatic
6. Сохраните изменения

### GoDaddy

1. Войдите в аккаунт
2. Перейдите в My Products → Domains
3. Нажмите DNS рядом с txd.bike
4. Добавьте новую запись:
   - Type: A
   - Name: @
   - Value: 76.76.21.21
   - TTL: 600 seconds
5. Сохраните

### Cloudflare

1. Войдите в аккаунт
2. Выберите домен txd.bike
3. Перейдите в DNS → Records
4. Добавьте A запись:
   - Type: A
   - Name: @
   - IPv4 address: 76.76.21.21
   - Proxy status: DNS only (серый облачко)
   - TTL: Auto
5. Сохраните

### Google Domains

1. Войдите в аккаунт
2. Выберите домен txd.bike
3. Перейдите в DNS
4. Добавьте запись:
   - Type: A
   - Name: @
   - Data: 76.76.21.21
   - TTL: 3600
5. Сохраните

## Проверка настройки

После добавления записи проверьте:

```bash
# Проверка A-записи
dig txd.bike A +short
# Должно вернуть: 76.76.21.21

# Или через nslookup
nslookup txd.bike
```

## Время распространения DNS

- Обычно: 5-30 минут
- Максимум: до 48 часов (редко)

## Проверка в Vercel

После настройки DNS Vercel автоматически проверит домен и отправит email при завершении верификации.

Проверить статус можно командой:

```bash
npx vercel domains inspect txd.bike
```

## Дополнительные записи (если нужны)

### Для поддомена www

```
Тип: CNAME
Имя: www
Значение: cname.vercel-dns.com
```

### Для всех поддоменов (wildcard)

```
Тип: CNAME
Имя: *
Значение: cname.vercel-dns.com
```

## Важно

- Не добавляйте A-запись для www поддомена, используйте CNAME
- После изменения DNS подождите несколько минут перед проверкой
- Убедитесь, что домен не имеет других конфликтующих записей
