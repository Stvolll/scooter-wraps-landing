# Настройка DNS для txd.bike в Namecheap

## Пошаговая инструкция

### Шаг 1: Вход в аккаунт Namecheap

1. Откройте https://www.namecheap.com/
2. Войдите в свой аккаунт

### Шаг 2: Переход к управлению доменом

1. В верхнем меню нажмите **"Domain List"** или перейдите по ссылке: https://www.namecheap.com/domains/list/
2. Найдите домен **txd.bike** в списке
3. Нажмите кнопку **"Manage"** справа от домена

### Шаг 3: Настройка DNS записей

1. В разделе **"Domain"** найдите секцию **"Advanced DNS"**
2. Прокрутите вниз до секции **"Host Records"**

### Шаг 4: Добавление A-записи для основного домена

**Если у вас уже есть A-запись для @:**

- Нажмите на иконку редактирования (карандаш) рядом с существующей A-записью
- Измените значение на: `76.76.21.21`
- Нажмите **"Save"** (галочка)

**Если A-записи для @ нет:**

1. Нажмите кнопку **"Add New Record"**
2. Выберите тип: **"A Record"**
3. Заполните поля:
   - **Host**: `@`
   - **Value**: `76.76.21.21`
   - **TTL**: `Automatic` (или `3600`)
4. Нажмите **"Save"** (галочка)

### Шаг 5: Добавление CNAME для www поддомена (опционально)

1. Нажмите **"Add New Record"**
2. Выберите тип: **"CNAME Record"**
3. Заполните поля:
   - **Host**: `www`
   - **Value**: `cname.vercel-dns.com`
   - **TTL**: `Automatic` (или `3600`)
4. Нажмите **"Save"** (галочка)

### Шаг 6: Удаление лишних записей (если есть)

Убедитесь, что нет конфликтующих записей:

- Удалите старые A-записи с другими IP-адресами для @
- Удалите старые CNAME записи для www, если они указывают на другие домены

### Шаг 7: Сохранение изменений

1. Все изменения сохраняются автоматически при нажатии на галочку
2. Дождитесь сообщения об успешном сохранении

## Итоговые записи должны выглядеть так:

```
Type    Host    Value                TTL
A       @       76.76.21.21          Automatic
CNAME   www     cname.vercel-dns.com Automatic
```

## Проверка настройки

### Через терминал:

```bash
# Проверка A-записи
dig txd.bike A +short
# Должно вернуть: 76.76.21.21

# Проверка через nslookup
nslookup txd.bike
# Должно показать: 76.76.21.21
```

### Через онлайн-сервисы:

- https://dnschecker.org/#A/txd.bike
- https://www.whatsmydns.net/#A/txd.bike

## Время распространения DNS

- **Namecheap**: обычно 5-15 минут
- **Глобальное распространение**: до 30 минут
- **Максимум**: до 48 часов (редко)

## Важные замечания

1. **Не меняйте Nameservers** - оставьте стандартные Namecheap nameservers
2. **TTL**: Рекомендуется использовать "Automatic" для автоматической оптимизации
3. **Проверка**: После сохранения подождите 5-10 минут перед проверкой
4. **Email от Vercel**: После настройки DNS Vercel автоматически проверит домен и отправит email

## Troubleshooting

### Если домен не работает после настройки:

1. **Проверьте правильность записи:**
   - Host должен быть `@` (не `txd.bike` или пусто)
   - Value должен быть точно `76.76.21.21`

2. **Очистите DNS кэш:**

   ```bash
   # macOS/Linux
   sudo dscacheutil -flushcache; sudo killall -HUP mDNSResponder

   # Или просто подождите 10-15 минут
   ```

3. **Проверьте через разные DNS серверы:**
   - Google DNS: 8.8.8.8
   - Cloudflare DNS: 1.1.1.1

4. **Убедитесь, что нет конфликтующих записей:**
   - Удалите все старые A-записи для @
   - Удалите старые CNAME для www (если они не указывают на Vercel)

## Следующие шаги

После настройки DNS:

1. Подождите 5-15 минут
2. Проверьте домен: `https://txd.bike`
3. Vercel автоматически проверит и активирует домен
4. Вы получите email от Vercel о завершении верификации
